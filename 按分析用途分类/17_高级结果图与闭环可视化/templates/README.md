# Templates

Recommended reusable templates to port into active projects:

- patient-level risk landscape: patient order + predicted probability + true label + feature heatmap + SHAP heatmap.
- response circuit circos: cell-state evidence tracks plus communication links.
- target evidence wheel: target perturbation burden plus drug/external support plus cell-state links.
- ligand-target-response circuit: sender cell -> ligand -> receiver state -> target gene program -> phenotype.
- spatial niche response map: spatial cell-state score + niche graph + communication vector field.

Keep templates project-specific: copy only the needed script into a project and adapt input paths explicitly.
