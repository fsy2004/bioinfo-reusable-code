synthetic, for demo only — 582 D-SPIN 模块示例数据
=================================================

由脚本生成(种子 582),不是真实实验数据,请勿用于任何生物学推断。

生成过程:
1. 定义 8 个基因程序之间的真实耦合矩阵 J_true(两个正耦合模块 + 两条跨模块负耦合);
2. 定义 5 个扰动各自的真实响应场 h_true(每个扰动直接作用 1-2 个程序);
3. 对每个 (扰动 × batch) 样本用 Gibbs 采样从 Ising 模型抽 150 个细胞的 ±1 程序状态
   (6 个条件 × 2 个 batch = 12 个样本);
4. 程序活性经基因载荷矩阵 W 映射为计数,Poisson 抽样后做 CP10K + log1p 归一化。

文件:
  expression.csv             1800 细胞 × 48 基因,log 归一化表达(行=细胞)
  cell_meta.csv              cell_id / sample_id / batch / if_control
  ground_truth_coupling.csv  8×8 真实耦合矩阵(仅供基线评估;真实数据没有它)
  ground_truth_response.csv  6×8 真实响应场(同上)

真值文件的存在正是本模块自带对照的意义:程序状态的原始相关网络会把
模块内间接共变也算成边,而自旋耦合矩阵试图把直接耦合从间接共变里拆出来。
有真值才谈得上"哪种更接近直接耦合"。
