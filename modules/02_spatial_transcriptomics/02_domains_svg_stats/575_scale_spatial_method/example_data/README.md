# example_data — synthetic, for demo only

`spatial_counts_synthetic.csv` 是**人工合成**数据,仅用于本模块的冒烟测试与展示图,不代表任何真实组织。

- 36×36 网格,1296 个 spot,80 个基因(负二项计数 + 高斯噪声,种子 200)。
- 结构:3 个粗域(沿 y 分带)× 每域 3 个细亚域(沿 x 分段)= 9 个嵌套真域。
- 粗域信号强(每域 12 个标记基因,均值偏移 ~5.5),细亚域信号弱(每域 4 个基因,偏移 ~1.9),
  因此细结构只有在加入空间上下文后才稳定 —— 用来演示多尺度域识别。
- 真值列 `domain_coarse` / `domain_fine` 只用于评估,不参与聚类。
