# Templates

Reusable figure templates to port into active projects.

## Templates

- patient-level risk landscape: patient order, predicted probability, true label, feature heatmap, and SHAP heatmap.
- response circuit circos: cell-state evidence tracks with communication links.
- target evidence wheel: target perturbation burden, drug and external support, and cell-state links.
- ligand-target-response circuit: sender cell, ligand, receiver state, target gene program, phenotype.
- spatial niche response map: spatial cell-state score, niche graph, and communication vector field.

## Usage

Keep templates project-specific. Copy only the needed script into a project and adapt input paths explicitly.
